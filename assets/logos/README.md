Place the requested logos in this folder using the exact filenames listed below (SVG preferred for best sharpness):

Required filenames:

- github.svg
- python.svg
- javascript.svg
- html5.svg
- css3.svg
- kali.svg
- linux.svg
- nmap.svg
- burpsuite.svg
- wireshark.svg

Example curl commands (these are examples and may change if sources move):

# GitHub Mark

curl -L -o github.svg https://github.githubassets.com/images/modules/logos_page/GitHub-Mark.svg

# Python logo

curl -L -o python.svg https://www.python.org/static/community_logos/python-logo-master-v3-TM.svg

# JavaScript (from devicon repo)

curl -L -o javascript.svg https://raw.githubusercontent.com/devicons/devicon/master/icons/javascript/javascript-original.svg

# HTML5 & CSS3 (from devicon)

curl -L -o html5.svg https://raw.githubusercontent.com/devicons/devicon/master/icons/html5/html5-original.svg

# Linux (Tux) - example
